# 移动端适配说明

本文记录 v1.0.0 的移动端能力和后续开发需要保持的交互约束。

## 适配范围

| 模块 | 当前实现 |
|------|----------|
| 全局壳层 | 桌面固定侧栏；移动端抽屉导航和紧凑顶栏 |
| 对话 | 消息宽度、Markdown、输入区、底部标签和结晶弹窗适配窄屏 |
| 工作台 | 文件树在移动端使用抽屉，预览区独占可用宽度 |
| 文件管理 | 分类导航、文件列表、上传和预览弹窗支持触摸操作 |
| 关系图 | 单指拖拽节点、单指平移、双指缩放、双击节点预览 |
| 概况 | 统计卡片和内容区限制宽度，纵向滚动条保持在模块右侧 |
| 设置 | 顶部横向 Tab，内容区域独立滚动 |

## 断点

主要布局断点为 Tailwind `lg`，即 `1024px`：

- `< 1024px`：移动/平板布局。
- `>= 1024px`：桌面布局。

部分卡片和弹窗继续使用 `sm`、`md` 断点调整列数和最大宽度。

## 全局布局约束

移动端横向溢出问题通常不是最外层宽度造成，而是内部 Flex、Grid、表格、代码块或 SVG 子元素设置了不可收缩宽度。

开发页面或组件时保持：

```text
App Shell       w-full max-w-full min-w-0
主内容           flex-1 min-w-0 overflow-hidden
页面滚动区         min-h-0 overflow-auto
Flex 子项         min-w-0
长文本            break-words / break-all
代码块            局部 overflow-x-auto
```

不要通过给 `body` 增加大范围横向滚动解决单个模块溢出，否则会影响文件管理和对话等已经适配的页面。

## 对话模块

- 用户和助手气泡在窄屏下使用接近全宽布局。
- Markdown 表格、代码块和长 URL 只能在消息内部滚动或换行。
- 风险提示、耗时、时间和结晶操作允许分行，不应撑宽消息列表。
- 输入框移动端字号保持 16px，避免 iOS Safari 聚焦放大。
- 结晶弹窗使用视口宽高限制；正文区域滚动，外层保持固定布局。

## 知识库工作台

- 桌面文件树固定在左侧并支持调整宽度。
- 移动端文件树放入 Sheet，选择文件后自动关闭。
- 预览、编辑和反向链接区域必须设置 `min-w-0`。
- 文件预览弹窗使用真实文件路径；只有 Markdown 内部链接使用 Wiki 路径解析。

## 知识概况

- 外层模块宽度保持 `100%`，内部统计区使用响应式列数。
- 可能包含长路径的列表项必须允许文本断行。
- 纵向滚动由概况主内容区承担，不在单个卡片外再套整页滚动。

## 关系图触摸操作

移动端触摸逻辑集中在：

```text
frontend/src/components/wiki/useGraphTouchGestures.ts
```

它与桌面鼠标事件保持分离，避免修改触摸逻辑时破坏 PC 拖拽行为。

### 手势映射

| 手势 | 行为 |
|------|------|
| 单指按住节点并移动 | 拖拽节点，节点紧跟手指 |
| 单指在空白处移动 | 平移画布 |
| 双指移动 | 以双指中心缩放和平移 |
| 快速双击节点 | 打开节点预览，不先改变缩放倍率 |

### 事件约束

- 节点点击与拖拽通过移动阈值区分。
- 未发生位移的点击不能注入力学扰动。
- 节点拖拽优先于画布平移。
- 双击节点预览优先于双击缩放。
- 触摸结束后必须清理拖拽、高亮和半透明状态。
- 触摸事件不应写入桌面鼠标拖拽状态。

## 关系图性能

- 生长动画和力学计算通过 `requestAnimationFrame` 更新。
- 点数增长时动态放大逻辑画布，而不是持续放大 DOM 容器。
- 自动缩放使用线性插值，不使用弹簧或变速缓动。
- 用户主动缩放后暂停生长自动缩放，避免视图与手势争夺控制权。
- 搜索框进入编辑状态会终止渐进生长，立即展示当前结果并执行扩散收尾。

## 移动端回归清单

- [ ] 360px、390px、768px 宽度下页面没有整页横向滚动。
- [ ] 侧栏抽屉可以打开、切换页面和关闭。
- [ ] 对话发送、停止、历史切换和结晶弹窗可操作。
- [ ] Markdown 表格和代码不会撑宽整个页面。
- [ ] 工作台文件树抽屉选择文件后正确关闭。
- [ ] 文件管理可以点击、上传、预览和下载。
- [ ] 概况内容完整显示，滚动条位于正确位置。
- [ ] 关系图可以平移、缩放、拖拽节点和双击预览。
- [ ] 触摸图谱后切回桌面鼠标操作没有残留状态。
- [ ] 横竖屏切换后弹窗和关系图仍在可视范围。
