# 版本说明

本项目使用 `v主版本.次版本.修订版本` 的版本标签。

## [1.0.0] - 2026-07-16

LLM-Wiki UI 首个正式版本，提供知识库管理、关系图、Hermes 流式对话、对话结晶、用户权限和内网部署能力。

### 知识库工作台

- 按实体、主题、摘要和结晶分类浏览 Wiki 文件。
- 支持 Markdown 页面预览、编辑和保存。
- 支持页面 Wikilink 跳转与反向链接查看。
- 支持从当前页面进入关系图并聚焦对应节点。
- 桌面端文件树支持宽度调整，移动端使用抽屉文件树。

### 文件管理

- 浏览 `raw/originals`、`raw/fulltext` 和 `raw/inbox` 文件目录。
- 支持文件选择、拖放上传和文件夹目录结构保留。
- 上传队列最多同时执行三个任务，并展示进度、成功、失败和重试状态。
- 支持 PDF、Office、文本、Markdown、图片和压缩包等配置允许的格式。
- 使用上传清单记录文件路径、MD5、大小和上传时间，并检查重复内容。
- 展示原件的待处理、全文已提取和已生成 Wiki 阶段。
- 支持 `.txt`、`.md`、`.json`、代码和配置文件等文本内容在线预览。
- 支持二进制文件下载，以及文件和目录删除。

### 知识概况

- 展示原件、全文、实体、主题、摘要等数量统计。
- 展示原件到全文、全文到 Wiki 的处理管线状态。
- 展示待处理原件路径和 Wiki 分类分布。
- 展示 MD5 相同的重复文件组。
- 支持桌面端和移动端独立滚动与响应式卡片布局。

### 关系图

- 使用 SVG 展示二维力导向知识关系图。
- 支持实体、主题、摘要和其他节点分组显示。
- 支持全局图、局部图、节点聚焦和一级邻居高亮。
- 支持按关键词搜索节点并更新当前结果集。
- 支持节点组、孤立节点、标签、节点大小、边宽和力学参数设置。
- 支持节点生长动画、扩散收尾和按节点总数计算的动态画布。
- 支持自动缩放、缩放按钮、滚轮缩放、画布平移和重置视图。
- 节点拖拽时紧跟指针，一级邻居受力，远距离节点按图跳数逐级减半。
- 移动端支持单指拖拽节点、单指平移画布、双指缩放和双击节点预览。

### AI 对话

- 从 Hermes Gateway 获取可用模型并选择当前会话模型。
- 支持新建、切换、重命名、清空和删除本地对话会话。
- 每个本地会话使用独立 Hermes Session Key，隔离不同对话上下文。
- 支持 SSE 流式回复、处理步骤、停止生成和中断内容保存。
- 用户消息和助手消息持久化到 SQLite，刷新页面后可恢复历史记录。
- 助手回复展示年月日时间、实时生成耗时和完成后的持久耗时。
- 助手消息展示大模型内容提示和风险操作核对提示。
- Markdown 正文支持标题、列表、表格、代码块和链接。

### 对话结晶

- 助手回复气泡提供结晶入口和已结晶状态。
- 点击结晶后展示二次确认弹窗。
- 确认弹窗展示结晶主题、会话 ID、用户问题、完整助手回复和正文长度。
- 助手正文区域支持独立滚动，提交错误和重复信息在弹窗内展示。
- 浏览器通过 FastAPI BFF 提交，Webhook HMAC Secret 不暴露给前端。
- 支持按消息 ID 和规范化正文检查重复提交。
- 结晶主题不参与正文重复判断。
- 支持用户明确选择强制再次提交。
- Hermes 接收任务后返回异步提交状态，并由 Agent 写入知识库。

### 用户与权限

- 提供本地用户名密码登录、Access Token 和 Refresh Token。
- 提供管理员和普通用户默认账号。
- 管理员可以管理普通用户资料、密码、状态和模块权限。
- 权限覆盖工作台、文件管理、关系图、概况、对话和设置模块。
- 密码或权限变化后通过 `token_version` 使旧 Token 失效。
- 支持可选 Odoo JWT SSO 用户管理模式。
- Odoo 模式支持用户自动创建、账号来源识别和只读用户列表。

### 设置与帮助

- 支持亮色、暗色和跟随系统主题。
- 支持通知偏好、账号信息、退出登录和版本信息展示。
- 提供 LLM-Wiki 显示相关设置。
- 帮助页面按本地账号模式或 Odoo 模式展示对应使用说明。
- 左侧侧栏展示当前应用版本号。

### 移动端与响应式

- 主侧栏在移动端使用抽屉导航。
- 对话消息、Markdown 内容、输入区和结晶弹窗适配窄屏。
- 工作台文件树、文件管理、知识概况和设置页面支持触摸操作。
- 关系图触摸手势与桌面鼠标事件分离。
- 页面滚动和模块内部滚动按功能区域独立控制。

### 后端与数据

- FastAPI 提供认证、用户、权限、知识库、上传、关系图、对话和结晶 API。
- SQLite 保存用户、Token、权限、对话会话、消息和结晶提交记录。
- 知识库文件通过共享 `knowledge-base` 目录与 `hermes-data` 协同工作。
- 上传清单在启动时补录真实原件并清理文件已不存在的记录。
- Wiki 同名标题按实体、主题、摘要和其他路径的固定优先级解析。
- OpenAPI 文档通过 `/api/docs` 提供。

### 部署

- 提供 React、nginx 和 FastAPI BFF 的 Docker 单镜像部署。
- Docker Compose 持久化 SQLite，并挂载 Hermes 共享知识库。
- 支持本地开发、内网生产、离线镜像交付、版本升级、数据备份和回滚。
- 对话通过 Hermes Gateway `8642`，结晶通过 Hermes Webhook `8644`。

### 版本约定

- 前端 API 默认使用同源 `/api`，特殊部署可在构建时设置 `VITE_API_BASE`。
- SQLite 数据库、数据库备份、前端构建产物和离线镜像 `.tar` 不进入源码版本控制。
- Wiki 编辑区提供页面阅读和编辑；文件删除统一从文件管理模块执行。
